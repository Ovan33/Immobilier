Detailed Description

Fenêtre de détail du client.

Cette classe permet d'afficher une fenêtre de détail du client. Il est utile à la création, mais aussi à la modification des données du client telles que son nom ou son adresse. La fenêtre client permet également de sauvegarder des modifications faites sur le client affiché.
