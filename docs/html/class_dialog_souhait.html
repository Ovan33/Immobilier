Detailed Description

Écran de détail d'un souhait.

Cette classe permet d'afficher un souhait et les données associées à ce souhait. Il est alors possible de modifier ces données et de sauvegarder le souhait. Cet écran est également utilisé lors de la création d'un souhait.
