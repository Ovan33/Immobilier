Detailed Description

Fenêtre d'accueil du logiciel.

L'application est organisée autour d'un certain nombre de fenêtre dépendantes les unes des autres. La classe dialogaccueil contient la fenêtre principale. Toutes les autres fenêtres et tous les autres objets dépendent d'elle. Sa destruction entraîne la destruction de tous les autres objets et la fermeture de l'application.
